# CSS Basics Practice Project

A beginner-friendly project created to practice and demonstrate essential CSS concepts using clean, readable, accessible, and SEO-friendly HTML.

---

##  Topics Covered

- CSS Outline  
- CSS Text Properties  
- CSS Fonts  
- CSS Links  
- CSS Lists  
- CSS Tables  
- CSS Icons (Font Awesome)  
- CSS Position  
- CSS Max-Width  
- CSS Display  

---

##  Technologies Used

- HTML5 (Semantic & Accessible)
- CSS3 (External Stylesheet)
- Font Awesome Icons

---

---

##  Key Features

- SEO-optimized HTML structure  
- Accessibility-friendly markup  
- Clean and readable CSS  
- Beginner-focused learning layout  
- Ready for GitHub portfolio  

---

##  How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Explore different CSS features section by section
