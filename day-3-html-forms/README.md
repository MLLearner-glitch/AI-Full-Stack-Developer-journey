## What I learned

- Importance of semantic HTML structure

## Form Attributes
 - action -> Specifies the URL where the form data will be sent after submission
 - method -> Specifies How the form data is sent to the server
 - name -> Identifies the form data when it is sent to the server
 - id -> Unique identifier for the form
