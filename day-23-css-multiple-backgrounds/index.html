Day 23 - CSS Multiple Backgrounds

CSS allows you to add multiple Background images for an element,through the Background-image property.

The different background images are separated by commas,and the images are stacked at top of each other,where the first image is closest to the viewer.

The Multiple background images can be specified using either the individual background properties with the background shorthand property.

background-origin:

The CSS background-origin property specifies where the background is positioned.

This property can have one of the following values:

  • border-box - The background image starts from the top-left corner of the border edge.
  • padding-box - The background image starts from the top-left corner of the padding edge.This is default.
  • content-box - The background image starts from the top-left corner of the content edge.

Note: This property has no effect if the background-attachment property is set to fixed.

Example:

background-origin:padding-box
