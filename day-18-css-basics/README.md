# Day 18 - CSS Practice Project

This project is a practice/demo for various **CSS topics** learned on Day 18.

---

## Topics Covered

- CSS Combinators
- Pseudo-classes
- Pseudo-elements
- Opacity
- Navigation Bar
- Dropdown Menu
- Image Gallery
- Image Sprites
- Attribute Selectors

---

## Project Description

This project demonstrates intermediate-level CSS concepts using a single HTML page and external CSS.

Each section in the webpage focuses on a specific CSS concept:
- Navigation bar with hover-based dropdown
- Combinator selectors using parent and child elements
- Pseudo-classes like `:hover`
- Pseudo-elements like `::before`
- Opacity effects
- Image gallery with hover zoom
- Image sprites using `background-position`
- Attribute selectors for form inputs

---

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Make sure images are placed inside the `images` folder.

---

## Notes

- External CSS is used for better readability and maintainability.
- Images can be replaced with your own if required.
- This project is created for **learning and practice purposes**.
