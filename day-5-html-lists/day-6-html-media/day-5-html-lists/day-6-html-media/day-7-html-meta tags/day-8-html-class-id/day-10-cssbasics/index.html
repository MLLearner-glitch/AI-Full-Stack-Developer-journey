Day 10 - CSS Basics

What is CSS?

CSS is the language we use to style a web page: 

CSS Stands for Cascading Style Sheets
CSS describes how HTML elements are to be diplayed on screen, page or other media
CSS saves a lot of work.it can control the layout of multiple web pages all at once
External stylesheets are stored in .css files

Why use CSS?

CSS is used to define styles for your web pages,including the design, layout and variations in display for different devices and screen sizes.

CSS Syntax

A CSS rule consists of a selector and a declaration block:

h1 { color:blue; font-size:10px;} 

Here, h1 is a selector
color:blue; is a declaration and also known color as property and blue as value.

The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by a semicolon(;). Each declaration includes a css property name and a value separated by colon(:).

CSS Selectors

CSS Selectors are used to find or select the HTML element you want to style.

We can divide CSS selectors into five categories: 

1.Simple Selector(select elements based on name, id, class)
2.Combinator Selectors(select elements based on specific relationship between them)
3.Pseudo-class selectors(select elements based on certain state)
4.Pseudo-Element selectors(select and style a part of an element)
5.Attributes Selectors(select elements based on attribute or attribute values

The CSS Universal Selector: 

Universal Selector (*) selects all HTML elements on the page.

The CSS Grouping Selector: 

To group selectors, seperate each selector with a comma(,)

How To Add CSS?

When a browser reads a stylesheet, it will format the HTML document according to the information in the stylesheet.

There are THREE ways of inserting a style sheets; 

1.Inline CSS
2.Internal CSS
3.External CSS

1.Inline CSS: 

Inline style may be used to apply a unique style for a single element.
To use inline styles,add the style attribute to the relevant element.The style attribute can contain any CSS property.

2.Internal CSS: 

An Internal stylesheets may be used if one single HTML page has a unique style.
The internal stylesheet is defined inside the style section inside the headsection.

3.External CSS: 

With an external stylesheets you can change the entire web page by changing just one file.
Each HTML page must include reference to the external style sheets file inside the  element inside the head section.

Comments

comments are used to explain the CSS code,and may help when you edit the source code at a later date.comments are ignored by browsers.

CSS Errors

Errors in CSS can lead to unexpected behaviour or style not being appiled correctly.
